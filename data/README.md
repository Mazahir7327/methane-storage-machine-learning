# Dataset Description

This directory contains datasets used for methane storage suitability assessment using machine learning approaches.

## Dataset Overview

The dataset consists of geological and reservoir parameters controlling methane storage potential in saline and depleted gas reservoirs.

The considered parameters include:

- Porosity
- Permeability
- Reservoir thickness
- Pressure
- Temperature
- Gas saturation
- Other relevant reservoir properties

## Data Usage

The dataset is used for:

1. Reservoir parameter assessment
2. Machine learning model development
3. Feature importance analysis
4. Correlation analysis
5. Model performance evaluation

## Data Processing

Before model training, the dataset undergoes:

- Data cleaning
- Missing value handling
- Feature preparation
- Model input formatting

## Reproducibility

The provided scripts can be applied to compatible reservoir datasets following the same data structure.
