# Examples

This directory contains example workflows and quick tests for the methane storage machine learning framework.

## Example Contents

The examples include:

- Reservoir suitability assessment examples
- Machine learning prediction demonstrations
- Model performance evaluation examples
- Visualization examples
- Environment verification test

## Purpose

These examples provide guidance for applying the developed workflow to methane storage assessment problems using geological parameters and machine learning approaches.

## Quick Test

A quick test script is provided to verify that the required Python environment and machine learning dependencies are correctly installed.

The test file is:

quick_test.py

Run the quick test using:

python examples/quick_test.py

A successful execution will display the installed versions of the required Python libraries, confirming that the computational environment is properly configured.
