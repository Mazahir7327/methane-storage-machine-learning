# Dataset Description

This directory contains the dataset information used for machine learning-based methane storage suitability assessment.

## Dataset Overview

The dataset consists of geological and reservoir parameters controlling methane storage potential in saline and depleted gas reservoirs.

The considered parameters include:

- Porosity
- Permeability
- Reservoir thickness
- Pressure
- Temperature
- Gas saturation
- Other relevant reservoir properties

## Data Usage

The dataset is used for:

1. Reservoir parameter assessment
2. Machine learning model development
3. Feature importance analysis
4. Correlation analysis
5. Model performance evaluation

## Machine Learning Applications

The dataset supports the implementation of:

- Random Forest
- XGBoost
- Support Vector Machine (SVM)

for methane storage suitability prediction and classification.

## Data Availability

The original experimental/field dataset can be provided upon reasonable request.
